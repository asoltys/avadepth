Le rapport sur la fenêtre de circulation montre la profondeur disponible pour la totalité du chenal pour une « fenêtre » de durée précise au cours de chaque cycle de marée. Pour produire le rapport,

  • Introduire une date. Cliquer sur la date texte pour choisir dans un calendrier.
  • Cliquer sur le bouton Débits

Avadepth montre les options de débit. À savoir : prévu, réel, défini par l'utilisateur ou à sélectionner dans la liste.

  • Choisir un rapport pour 1 jour ou 1 semaine.
  • Préciser la durée de la fenêtre pour laquelle les profondeurs doivent être calculées.
  • Sélectionner la navigation à une voie ou la navigation à deux voies.
  • Sélectionner le débit.
  • Cliquer sur le bouton Rapport de circulation pour commencer.

Avadepth montre le rapport de la fenêtre de circulation maritime. Pour imprimer le rapport :

  • Dans le menu du navigateur, sélectionner Fichier|Imprimer.
