Help - Transit Window

The Transit Window report shows the depth available for the entire channel for a window of specified length in each tide cycle. To generate the report,

  • Enter a date.  Click on the date text to choose using a calendar.
  • Click on the Flows button
Avadepth displays the options to use a flow rate.   These are predicted, actual, user-defined or select from the list.
  • Select to report on 1 day or 1 week.
  • Specify the Window Length for which depths are to be calculated.
  • Select one-lane or two-lane operation.
  • Select the flow rate to be used.
  • Click the Transit Report button to begin.

Avadepth displays the navigation transit window report.   To print the report to your printer:

  • In your browser menu, you can select File|Print to print a hard-copy.
