EVENT_TYPES

{Array(String)} Supported application event types.

Register a listener for a particular event with the following syntax

clickablelabel.events.register(type, obj, listener);

Supported ClickableLabel event types

labelClickedTriggered when the label is clicked.  Passes the object reference to the listeners.
labelTextChangedTriggered when the label text is changed.
