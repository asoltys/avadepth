Help - Survey Drawings

This section contains a list of all Fraser River survey drawings available in digital format.These drawing a\can all be view online after downloading the AutoCad WHIP Viewer.

The first thing you must do is select a River.

  • Click on the drop down menu and select a River.
  • Select a search method - use a map to select the coverage area for the drawings or
  • Select an area of the River using feature name ex. Sandheads Reach.
  • Click on the Next button.

If you chose to use a map and if there is a map available it will appear in the window on the right.

  • Click on the Current radio button to select only the most recent drawing for the area or
  • Click on All to get all the drawings for the chosen area.
  • Select the type of drawing you would like to get to help narrow the search - leave blank for all types.
  • Click on a red tile to select an area on the river to get the drawings bringing up a more detailed image.
  • Select a small red box to bring up the drawings for the area of the River or
  • Click elsewhere on the image to bring up an overview drawin g on the larger area.

To select drawing using a feature name.

  • Select Recent or All drawings
  • Select Main or Secondary Channel drawings. Other is for structure drawings.
  • Select location from the list box of names.
  • Select the Type of drawing, leave blank for all Types
  • Click the Search Button

To view a drawing.

  • Click on the Drawing name in the table that is drawn in the window.

To print the graph to your printer:

  • In your browser menu, you can select File|Print to print a hard-copy.
